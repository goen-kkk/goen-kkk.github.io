Online Learning

  1. 基本的机器学习策略，错误驱动，一次只能学习一个样本
  2. 学习目标：减小错误率
  3. 学习过程
    1. 学习器遇到一个样本
    2. 做出一个预测
    3. 如果预测与实际不一致，则学习器进行更新
    4. 按照顺序，循序的学习，不断的去修正模型，进行优化

  • Consistent learner

    • 假设空间（hypothesis class），
    • （version space，解释空间，第t步时的假设空间）
    • 对于样本，随机地从取出一个函数进行预测，得到预测值
    • 如果预测与实际不一致，那么将从假设空间中去除。
    • 我们得到，即
    1. 最多会犯次错误

  • Halving learner

    • 初始化整个假设空间
    • 对于样本，使用中的每一个函数h进行预测，选取多数函数预测的标记作为最终的预测结果。 如果多数函数预测为0，则最终的预测结果就是0，反之亦然。
    • 如果预测与实际不一致，则将所有预测错误的从假设空间中去除

Probably Approximately Correct（PCA）Learning

  1. 输入空间（input space） - 所有的样本（examples or instances）
  2. 目标值的集合 - y 表示目标值（target）
  3. 概念(concept) c, 在0-1分类问题中， c 可以是的子集，其对应的 y=1。（参考书里用的是f）
  4. Concept class ：所有concept的集合
  5. 假定样本的分布符合i.i.d.
  6. Learner 的目的则是在所有可能的假设空间中,针对已有的符合概率分布的数据和对应的目标值进行学习，以希望得到一个最优的target concept

example:

一个教室里有一个教授（target concept）和一个研究生（learner），一群学生（可以是有限多个或者无限多个）陆续进入教室（X ），这些学生来自不同的专业 （Y），可能是医学、物理、数学、法律、计算机等等。每个学生有不同的特征，比如身高、长相、是否穿实验服等等，而这些特征符合一定的概率分布 ，每个学生之间的特征互相独立。Learner (研究生) 的目的是建立一个假设的集合（hypothesis space），根据每个学生的feature进行训练。进入教室的前N个学生都会告诉研究生他们是不是计算机专业，之后的学生的专业不告诉他/她。而教授则知道所有进入教室的学生是否是计算机专业。我们感兴趣的问题就具体化为需要多少个学生进入这个教室，研究生才能具备分辨他们专业的能力。


Error of Hypothesis

  • Generalization error/true error 泛化误差

    泛化误差是用来衡量一个学习机器推广未知数据的能力

    训练过程中的误差，就是训练误差。训练完毕、调参完毕的模型，在新的测试集上的误差，就是测试误差。假如所有的数据来自一个整体，模型在这个整体上的误差，就是泛化误差。通常说来，测试误差的平均值或者说期望就是泛化误差。泛化误差是衡量一个模型推广能力的标准，指示泛化性能。泛化误差越小，表示模型的推广能力越强，泛化性能越优。

PAC Learning

  • Learner不知道D（概率分布）和f（未知样本的实际标签）
  • accuracy 参数ϵ 和confidence参数δ
  • 训练数据的样本数量取决于ϵ 和δ的值，而不取决于D和f
  • learner会输出假设ℎ，其真实误差的概率为1−δ
  • Probably (with probability at least 1 − δ) Approximately (up to accuracy ϵ) Correct

PCA Learnability

  • The accuracy parameter ϵ determines how far the output classier can be from the optimal one (this corresponds to the “approximately correct")
  • A confidence parameter indicating how likely the classier is to meet that accuracy requirement (corresponds to the “probably" part of “PAC").

VC维度(VC dimension)

VC理论至少包括四个部分：

  • 学习过程一致性（consistency）理论：基于经验风险最小化原则，学习过程一致性的（必要且充分的）条件是什么？
  • 学习过程收敛速度的非渐近（Nonasymptotic）理论：学习过程的收敛速度有多快？
  • 控制学习过程泛化能力（generalization）的理论：如何控制学习过程的收敛速度（泛化能力）？
  • 构建学习机器的理论：如何构造可以控制泛化能力的算法？
