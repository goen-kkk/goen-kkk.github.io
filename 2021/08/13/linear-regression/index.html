线性回归

线性回归的思想：线性回归假定自变量（X）和因变量（Y）之间存在着线性关系，​

  • x is an explanatory variable/feature

  • y is the outcome/response

  • w is the slope

  • b is the intercept

    Goal：Find b and w so that for most points,

线性回归的损失函数

为了找到一个最佳拟合函数，我们需要定义一个损失函数，目标是该线性模型在输入的数据上损失最小

一个简单的例子：

最简单的想法就是最小化每个数据点到拟合线的垂直偏差的和

怎么处理 Categorical 变量

很多真实生活中的数据包含categorical 和 numerical features（分类特征和数字特征）

比如学生的专业有经济学、物理学和政治学，我们需要将类别编码为数字特征

  1. label encoding

    0 = Economics, 1 = Physics, 2 = Politics

    对于一个有m个category的特征，经过label encoding以后，每个category会映射到0到m-1之间的一个数。

    label encoding适用于ordinal feature （特征存在内在顺序）。

  2. One-hot encoding

    [1 0 0] = Economics, [0 1 0] = Physics, [0 0 1] = Politics

    对于一个有m个category的特征，经过独热编码（OHE）处理后，会变为m个二元特征，每个特征对应于一个category。这m个二元特征互斥，每次只有一个激活。

    稀疏编码，编码之后特征空间过大，一般用于category数量 <4的情况。
